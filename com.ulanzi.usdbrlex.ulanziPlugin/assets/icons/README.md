# Icon Requirements

This directory should contain the following icon files:

## Required Icons:
- **icon.png** (72x72px) - Main plugin icon displayed on deck buttons
- **actionIcon.png** (20x20px) - Small action icon for UI elements  
- **categoryIcon.png** (28x28px) - Category icon for plugin grouping

## Design Guidelines:
- Background color: #282828 (UlanziDeck standard)
- Use currency symbols ($ and R$) or exchange arrows
- Keep design simple and readable at small sizes
- Use white/light colors for visibility on dark background

## Temporary Solution:
For development purposes, you can copy icon files from the analogclock demo plugin and modify them, or create simple placeholder images with the currency symbols.
